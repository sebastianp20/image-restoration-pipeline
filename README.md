Setup
1. Clone this repo
2. Install dependencies: pip install -r requirements.txt
3. Download GoPro dataset from Kaggle
Project Structure
notebooks/ -main project notebook
src/ - modular Python scripts
data/ - raw and processed datasets
outputs/ - deblurred images, detections, metric
ai_logs/ - AI prompt and output attributions
